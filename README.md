
Blooper 
====
a simple VST plugin that record and loops audio in sync


Main features
===
Strong OSC interface
Shared metronome among instances of plugin
Quantization of slave instances


Quick Overview
====
once plugins added where you want audio to be recorded in your DAW, 
you can seamlessly record and loop any of these with OSC.
A master track (i.e : indexed 0) sets the master Tempo, then action on others (i.e rec / play...) can be quantized to ensure sync with the master
